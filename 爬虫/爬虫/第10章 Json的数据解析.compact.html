json数据解析

一、json数据简介

JSON是一种非常重要的数据格式，它并不是编程语言，而是一种可以在服务器和客户端之间传输的数据格式。

JSON（JavaScript Object Notation, JS对象简谱）是一种轻量级的数据交换格式。它基于 ECMAScript（European Computer Manufacturers Association, 欧洲计算机协会制定的js规范）的一个子集，采用完全独立于编程语言的文本格式来存储和表示数据。

简洁和清晰的层次结构使得 JSON 成为理想的数据交换语言。 易于人阅读和编写，同时也易于机器解析和生成，并有效地提升网络传输效率。

 

二、JSON三种格式

JSON的顶层支持三种类型的值：

  • 简单值：数字（Number）、字符串（String，不支持单引号）、布尔类型（Boolean）、null类型；

  • 对象值：由key、value组成，key是字符串类型，并且必须添加双引号，值可以是简单值、对象值、数组值；

    xxxxxxxxxx
    {
    "key1":  "value1",
    "key2": "value2",
    ....
    }

     

  • 数组值：数组的值可以是简单值、对象值、数组值；

    xxxxxxxxxx
    [
    {"key1": "value1"},
    {"key2": "value2"}
    ]

 

实例

xxxxxxxxxx
#json对象
{
  "name": "hhh",
  "age": 18,
  "friend": {
    "name": "sam"
  }
}
#json数组
[
  234,
  "abc",
  {
    "name": "amy"
  }
]

 

三、python的json解析

由于json格式是一种字符串格式，处理获取数据不方便，一般我们会将json转换为python中的列表或者字典来处理

使用 JSON 函数需要导入 json 库：import json。

 

json.dumps

json.dumps 用于将 Python 对象编码成 JSON 字符串。

xxxxxxxxxx
import json


data = [ { 'a' : 1, 'b' : 2, 'c' : 3, 'd' : 4, 'e' : 5 } ] #列表


data2 = json.dumps(data) #转为json字符串
print(data2)
print(type(data2))
xxxxxxxxxx
[{"a": 1, "b": 2, "c": 3, "d": 4, "e": 5}]
<class 'str'>

 

json.loads

json.loads 用于解码 JSON 数据。该函数返回 Python 字段的数据类型。

xxxxxxxxxx
import json


jsonData = '{"a":1,"b":2,"c":3,"d":4,"e":5}' #json字符串


text = json.loads(jsonData) #转换为列表或者字典
print(text)
print(type(text))
xxxxxxxxxx
{'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}
<class 'dict'>

 

 

四、综合案例

案例一：北京新发地数据解析

xxxxxxxxxx
import requests
import json
import csv


url = ' http://www.xinfadi.com.cn/getPriceData.html'


#参数
da={
    "limit":"20",
    "current":"1",
    "pubDateStartTime":"",
    "pubDateEndTime":"",
    "prodPcatid":"1186",
    "prodCatid":"1199",
    "prodName":""
}


#反扒头
dict = {
"User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:103.0) Gecko/20100101 Firefox/103.0"
}


resp = requests.post(url,data=da,headers=dict)
resp.encoding="utf-8"


data = json.loads(resp.text)


results = []
for item in data.get("list"):
    results.append([item.get("prodName"),item["prodCat"],item["prodPcat"],item["lowPrice"],item["highPrice"],item["avgPrice"]])




with open("新发地.csv","w",encoding="utf-8",newline="") as f:
    w = csv.writer(f)
    w.writerows(results)


resp.close()

 

案例二 豆瓣电影排行榜

xxxxxxxxxx
import requests
import json
import csv






url = ' https://movie.douban.com/j/chart/top_list'


#参数
data={
    "type":"17",
    "interval_id":"100:90",
    "action":"",
    "start":0,
    "limit":"20"
}


#反扒头
dict = {
"User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:103.0) Gecko/20100101 Firefox/103.0"
}


results = []


#循环拉取10页数据
for i in range(10):


    resp = requests.get(url,params=data,headers=dict)
    resp.encoding="utf-8"


    lst = json.loads(resp.text) #转为列表


    for item in lst:
        results.append([item.get("title"),item.get("score"),item.get("url")])


    data["start"] = data.get("start", 0) + 20


with open("豆瓣.csv","w",encoding="utf-8",newline="") as f:
    w = csv.writer(f)
    w.writerows(results)




resp.close()
